# 🚀 Getting Started

Follow the steps below to run Zingo on your local machine 👇

### 1️⃣ Clone or Download the Repository
Download the complete repository and open it in your preferred IDE.

### 2️⃣ Install Dependencies
Before running the app, install all required npm packages.

Open the terminal inside the project folder and run:

```bash
npm install
```
### 3️⃣ Enable Proxy Access
Zingo uses a proxy server to handle CORS (Cross-Origin Resource Sharing) issues while fetching APIs.

Before running the project, visit the link below and request temporary access to the proxy server:

```bash
🔗 https://cors-anywhere.herokuapp.com/corsdemo
```
> ⚠️ This step is required. Without enabling proxy access, some API requests may fail because of browser CORS restrictions.

### 4️⃣ Start the Development Server
After installing dependencies and enabling proxy access, start the project with:

```bash
npm run dev
```

## 🚀 Overview

Zingo is a next-generation food delivery platform that reimagines the traditional ordering experience by combining real-time data, intelligent UI/UX, and performance-driven architecture.

Built by integrating live vendor data (inspired by Swiggy’s API ecosystem), Zingo goes beyond basic food ordering by introducing advanced features like fitness-focused meal insights, high-performance filtering, and a seamless multi-cart system.

---

## 💡 What Makes Zingo Different?

Zingo is not just another food delivery app — it focuses on speed, personalization, and intelligent user experience.

- ⚡ Faster discovery of food items  
- 🧠 Smarter filtering and search  
- 💪 Health-focused meal insights  
- 🛒 Seamless multi-vendor ordering

## 🔥 Key Features

### 1) ZingoFit (Gym Mode)

- Dedicated fitness-focused food section  
- Dynamic calorie calculator  
- Nutritional breakdown including:
  - Protein  
  - Carbohydrates  
  - Fats  
- Covers 60+ gym-specific meals  

---

### 2) Smart Filters & Search

- GSAP-powered Veg / Non-Veg animated filters  
- High-performance search using JavaScript Map  
- Eliminates duplicate entries for cleaner results  
- ⚡ Improves discovery speed by ~40%  
- Supports filtering across 1000+ dishes instantly  

---

### 3) Advanced Multi-Cart System (Redux)

**Supports:**
- Add / Remove items  
- Increment / Decrement quantity  
- Multiple carts across different vendors  

**Integrated redux-persist for:**
- 💾 100% cart retention across sessions  

- Enables seamless multi-restaurant ordering  

---

### 4) Optimized Data Caching

- Implemented Redux-based caching

- Ensures smoother and faster user experience

### 5) Immersive UI/UX
